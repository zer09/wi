# Wi

Wi is a local, single-user, Linux-only browser-based coding-agent harness. This repository currently contains the Milestone 0 workspace bootstrap and architecture guardrails for v0.1.0.

## Requirements

- Linux
- Node.js 24 LTS
- pnpm 11

## Bootstrap

```sh
pnpm install
pnpm check
```

See [`docs/plans/v0.1-first-vertical-slice.md`](docs/plans/v0.1-first-vertical-slice.md) for the milestone plan and [`docs/adr/README.md`](docs/adr/README.md) for canonical architecture decisions.
